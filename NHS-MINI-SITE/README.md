# OTOS Continuity — Website & Materials

## About
Premium website and NHS pitch materials for **OTOS Continuity** — a pilot model connecting care, support and recovery services into one joined-up pathway for adults with undiagnosed ADHD in addiction settings.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main website (one-page long scroll) |
| `nhs-brief.html` | NHS / ICB one-page briefing document (print-ready) |
| `partner-brief.html` | Partner & Scott explainer document (print-ready) |
| `css/style.css` | All website styles |
| `js/main.js` | Scroll effects, animations, mobile nav |
